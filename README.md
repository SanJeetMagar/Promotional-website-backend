# Promotional-website
This is a promotional website for a fictional company called "Tech Solutions". The website is designed to showcase the company's services and products, as well as provide information about the company and its team.
## Features
- Responsive design that works on all devices
- Clean and modern layout
- Easy navigation with a clear menu
- Sections for services, products, about us, and contact information
- Contact form for users to get in touch with the company
